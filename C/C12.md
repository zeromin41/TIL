### 중첩 for문

for문안에 for문이 중첩되어 있는 구문

ex. 피라미드 출력

```c
#include <stdio.h>


int main(void)
{
	int a;
	int b;

	for (a = 0; a < 5; a++) 
	{
		for (b = 0; b <= a; b++) {
			printf("*");
		}
		printf("\n");
	}


	return 0;
}
```
```c
*
**
***
****
*****
```
a가 5보다 작을때 하나씩 증가하는 경우에, b는 a보다 작을때 하나씩 증가하며 *모양을 출력한다.

a는 0일때 b는 *을 0개 출력.

a는 1일때 b는 *을 0개 출력 + 1개 출력.

a는 2일때 b는 *을 0개 출력 + 1개 출력 + 2개 출력.

...

이런식으로 *이 출력됨.
